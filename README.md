# SquareToDo
ToDos reimagined!!! A react Native app built with Expo for 2 dimensional ToDos

To get started 

```
$git clone https://github.com/Vaibhav21pandit/SquareToDo/
```
then install all required packages in package.json with

```
$npm install
```
